  ```
  git init //用来初始化，创建git可以管理的仓库

  git remote add origin 网址 //网址提取github上项目的网址
  git push //完成提交

# 2.markdown的学习

```总述
感觉markdown的上手还是比git及github友好的，而且对于更多的人来说也更实用。下面我将陈述我学习的简单语法。
```

```语法简介
# 标题  //可以放大标题，其中 # 越多，标题越小，最多可累加到6级
** ** //加粗字体
== == //字体高亮
~~ ~~ //字体上有删除线
* * //字体斜体
>  //引用
--- //分割线
![照片名称]（图片地址） //插入图片
[地址命名](网址) //插入网址
-  //无序列表
```
